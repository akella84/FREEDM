The dispatcher and connection manager are shared with the modules.

The port is free to be bound to.
Postcondition:

void freedm::broker::CBroker::Run ( )

Run the Server's io_service loop.

Calls the ioservice run (initializing the ioservice thread) and then blocks until the ioservice runs out of work.

Precondition:
The ioservice has not been allocated a thread to operate on and has some schedule of jobs waiting to be performed (so it doesn't exit immediately.)
Postcondition:
The ioservice has terminated.
Returns:
none
void freedm::broker::CBroker::Stop ( )

Puts the stop request into the ioservice queue.

Registers a stop command into the io_service's job queue. when scheduled, the stop operation will terminate all running modules and cause the ioservice.run() command to exit.

Precondition:
The ioservice is running and processing tasks.
Postcondition:
The command to stop the ioservice has been placed in the service's task queue.
