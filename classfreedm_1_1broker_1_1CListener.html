Construct a CConnection with the given io_service.

Constructor for the CConnection object. Since the change to udp, this object can act either as a listener or sender (but not both) to have the object behave as a listener, Start() should be called on it.

Precondition:
An initialized socket is ready to be converted to a connection.
Postcondition:
A new CConnection object is initialized.
Parameters:
p_ioServiceThe socket to use for the connection.
p_managerThe related connection manager that tracks this object.
p_dispatchThe dispatcher responsible for applying read/write handlers to messages.
uuid,:The uuid this node connects to, or what listener.

void freedm::broker::CListener::Stop ( ) [virtual]

Stop all asynchronous operations associated with the CConnection.

Stops the socket and cancels the timeout timer. Does not need to be called on a listening connection (ie one that has had Start() called on it.

Precondition:
Any initialized CConnection object.
Postcondition:
The underlying socket is closed and the message timeout timer is cancelled.

Implements freedm::broker::CReliableConnection.
