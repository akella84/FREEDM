Chandy-Lamport snapshot algorithm

: Each node that wants to initiate the state collection records its local state and sends a marker message to all other peer nodes. Upon receiving a marker, peer nodes record their local states and record any message from other nodes until that node has recorded its state (these messages belong to the channel between the nodes).
